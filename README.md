🎵 Musify – Modern Web Music Player


Musify is a modern web-based music player designed with a clean glassmorphism interface and responsive layout. Users can upload, organize, search, favorite, and play local audio files directly within the browser.

Built using HTML, CSS, and JavaScript, Musify demonstrates modern front-end development concepts including responsive design, audio handling, local storage, and interactive UI components.

Whether you’re looking for a personal music player or a front-end portfolio project, Musify provides a solid foundation with an elegant user experience.

🎵 Music Library

* Upload local audio files
* Dynamic music library management
* Song listing with metadata
* Delete tracks from library
